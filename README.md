## Getting Started
 To get a local copy of this project, open your terminal and type:
  - git clone https://github.com/techwolf404/gofreelancer-backend.git
  - cd gofreelancer-backend
  - bundle install
  - rails s
 That's it! You can play with it as you wish 😄
